Guide

How to Use BetterTierListMaker:

Registration and Login:

  1. Click on the login link on the navigation bar.

  2. If you do not have an account, click the “Register Here” button.

  3. Enter a username and password for your account. Then click the register button.

  4. If it says your account was created successfully click the “login in here” link.

Using the Tier List Editor:

Importing Images and Dragging to Tier List Rows:
  1. Click the import images button.

  2. Click the “Choose Files” button.

  3. Select your images from your file explorer.

  4. Click the submit button.

  5. Your images should now appear at the bottom of the editor page.

  6. You can then drag your image to the tier list rows. You can move images around in their row and move images from row to row.

Edit Tier Name, Tier Background Color, Row Color, and Site Background Color:

  1. Click on the gear next to the row you want to edit.

  2. A prompt will appear at the top of your webpage which you ask you a set of questions regarding the settings you can change. If you want to change the settings in question you can click OK, otherwise click cancel.

    a. The first question is if you want to change the tier name.
    b. The second question is if you want to change the tier background color. You can use any CSS syntax to declare a color value. These include color keywords like red, green, and blue, hex codes like #ffffff, or rgb values like rgb(255, 255, 255).
    c. The same goes for row color and page background color.
